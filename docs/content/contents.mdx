Luxa includes compiled CSS files (ready to use), source CSS files (for customization), and build scripts. **No build tools required**—use the compiled files directly.

```sh
luxacss/
├── dist/
│   ├── compressed/
│   │   └── luxa.min.css
│   ├── expanded/
│   │   ├── luxa.css
│   │   └── luxa.css.map
│   └── individual/
│       ├── compressed/
│       │   └── [individual files].min.css
│       └── expanded/
│           └── [individual files].css
├── css/
│   ├── colors.css
│   ├── grid.css
│   ├── luxa.css
│   ├── reset.css
│   ├── spacing.css
│   ├── tokens.css
│   ├── typography.css
│   └── utilities.css
├── docs/
├── scripts/
│   ├── build.js
│   ├── build-framework.js
│   ├── dev.js
│   └── lint.js
├── postcss.config.js
├── license.md
├── package.json
└── readme.md
```
